# Cognitive Career & Job Recommendation System

## Overview
AI-based career recommendation system using cognitive reasoning and explainable AI to provide personalized career guidance with resume analysis and intelligent job matching.

## Quick Start

### Prerequisites
- Python 3.8+

### Setup
```bash
cd cognitive-career-recommender/backend
pip install -r requirements.txt
python simple_app.py
```

Access at `http://localhost:5000`

## Features
- NLP-based resume analysis
- Cognitive reasoning for recommendations
- Explainable AI insights
- Interactive dashboard with 9 career pages
- Skill gap analysis and learning roadmaps

## Tech Stack
- **Backend**: Flask, Python
- **AI/ML**: Scikit-learn, Pandas, NumPy
- **NLP**: spaCy, NLTK, TextBlob
- **Frontend**: Bootstrap 5, Font Awesome
- **Explainable AI**: SHAP, LIME

## Project Structure
```
backend/
├── simple_app.py          # Main application
├── ai_engine/             # Cognitive AI engine
├── nlp_processor/         # Resume analysis
├── utils/                 # Data processing
└── templates/             # HTML pages
```

## Dashboard Pages
- **Home** - Overview and statistics
- **My Profile** - User information
- **AI Analysis** - Skills and interests
- **Recommendations** - Career matches
- **Explainable AI** - Why these jobs
- **Skill Gap** - Learning paths
- **Roadmap** - 12-month plan
- **Reports** - Download insights
- **Settings** - Account options
